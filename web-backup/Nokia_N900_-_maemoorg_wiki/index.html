[edit] Camera Stills

Some features are only available with the latest firmware.

Still camera functionality
Feature Capability
Capture Resolutions 5 megapixel (2576x1936 pixels; 4:3), 3.5 megapixel (2560x1440 pixels; 16:9), 3 megapixel (2048x1536 pixels; 4:3), 1.3 megapixel (1280x960 pixels; 4:3)
Flash Dual LED
Lens Carl Zeiss optics, Tessar, 5.2 mm focal length, f/2.8, fixed aperture, focus 10 cm to infinity
Physical Features Hardware shutter button, with Lens Cover
Sensor CMOS 1/2.5" 31.2 mm focal length (35 mm equivalent)
Zoom 3X Digital
Socketed Replacable in case of damage or malfunction. In principle upgradable.

Sample pictures are available from the Flickr N900 Group.

[edit] Connectivity

Connectivity functionality
Feature Capability
Audio/Video Out Nokia AV 3.5 mm inc. TV Out (included cable CA-75U)
Bluetooth V2.1 +EDR with Following profiles A2DP,AVRCP,FTP,HFP,HSP,OPP (DUN PAN HID via customisation)
FM receiver RDS, US/EU/Japan bands, Headphone cable for antenna
FM transmitter Internal antenna, 88.1 to 107.9 MHz
GSM Quad-band EDGE 850/900/1800/1900
GPS A-GPS, GPS receiver, Network Positioning
Infra Red Port Community support of CIR from Irrecco project
WCDMA 900/1700/2100 MHz, Tmobile 2100 Mhz, U.S only
Micro USB (USB host mode in development) USB 2.0, USB Mass Storage, USB Charging
WiFi 802.11 b/g :WEP, WPA, WPA2 (AES/TKIP), UPnP, DLNA Certification

[edit] Memory

Main article: N900 filesystem


Memory specification
Feature Capability
Application memory 1.93 GB see the page on using /opt for applications
Expandable memory Micro SD slot for up-to 32 GB additional
Internal storage Maximum built in user storage 32 GB
System memory 1 GB (256 MB SDRAM plus 768 MB swap on EMMC)

The media player can use both the internal 32 GB and micro SD card as storage.
