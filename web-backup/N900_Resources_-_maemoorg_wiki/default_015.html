Maemo is a platform built on Linux and free desktop components with a UI optimized for mobile devices. Some components are developed openly in their respective upstream projects and some are developed here. (more)

Follow also in Qt Gitorious http://qt.gitorious.org/maemo-6-ui-framework
Check more Maemo community resources at http://maemo.org
Developer documentation and tools: http://www.forum.nokia.com/maemo
Maemo features and devices: http://maemo.nokia.com
