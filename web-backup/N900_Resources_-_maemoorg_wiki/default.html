eSpeak - Extra Russian and Chinese dictionaries 1.46.17

Extra espeak dictionaries for Russian, Mandarin Chinese and Cantonese.

This package enable espeak and applications using it, to speak in Russian, Mandarin Chinese and Cantonese.

See http://espeak.sourceforge.net/data/index.html for more information.

This package does nothing in itself. These dictionaries are packaged separately from espeak as they are quite large.
