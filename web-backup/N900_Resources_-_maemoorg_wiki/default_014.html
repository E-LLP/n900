Maemo 5 Certificate Manager

This project contains the certificate manager of the Maemo 5 platform. It consists of two source packages, maemo-security-certman for the libraries, command-line tools and the default root CA set; and maemo-security-certman-applet for the UI library and the control panel applet. Please consult the wiki (once the damn thing stops being created) for build instructions.

maemo-security-certman

The source package of libmaemosec, libmaemosec-certman, maemosec-certman-tools and maemosec-certman-common-ca packages.
