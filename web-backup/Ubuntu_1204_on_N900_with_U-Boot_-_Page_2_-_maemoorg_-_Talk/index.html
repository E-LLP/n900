Template for reports:
STATUS: Booted, system lockups from time to time (~ every third boot)
REVISION: 43, unchanged
MICROSD: 16 Gb, class 6
BOOTLOADER: U-Boot pr13
OPTIONS:
selected_image_source: ubuntu-daily
selected_firmware_source: local
dist_upgrade:
selected_kernel:
boot_parameters:
additional_packages:
And comments:
I will be updating options section with blank values when there will be more options available.
Use grep ^ * in the options/ folder to get them all at once. selected_microsd_device and fdisk_command_stack.file are irrelevant in most cases.
